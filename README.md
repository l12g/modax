[A tiny/powerful modal toolkit for web.](https://voidshub.github.io/modax/)

## Installation

```bash
npm i modax
# yarn add modax
```

## Usage

```js
import mdx form 'modax'

mdx.title('notice')
    .content("are u sure?")
    .show();
```
